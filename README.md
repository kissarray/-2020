# 云开发 quickstart
###########说明##########
一：此项目为云开发小程序
二：cloudfunctions为云函数
  2.1 需要上传并部署全部云函数
  2.2需要自建数据库
    code：为发布记录数据库
    counters：意见反馈数据库
    history：运动记录
    todos：打卡记录
    trees：为种树记录数据库
  2.3云开发环境
   env：very-xxxx
1-openapi开放接口,openapi下的cloudid为数据接口

2-storageconsole为文件上传接口

3-databasegu为数据库操作接口

4-一级接口
首页：index（包含天气api与视频api）
资讯： news（来自网上素材）
运动：sport（包含微信运动，天气api）
我的：my（个人信息及排行榜）
